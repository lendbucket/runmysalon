# Logo Rules — DO NOT CHANGE

The official Salon Envy® logo is at: public/images/logo-white.png
This is a white PNG with transparent background.
It must ALWAYS be displayed on dark backgrounds (#060e14 or darker).

NEVER:
- Replace with SVG text
- Replace with emoji or text
- Change the file path
- Add filters that change the appearance
- Display on light backgrounds

ALWAYS:
- Use <img src="/images/logo-white.png">
- Dark background behind it
- Width between 140px-220px depending on context
